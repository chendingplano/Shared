# Release 2026/02/14

1. EchoFactor GetUserInfoByUserID(...) retrieve user info from Kratos tables 
2. Added auth.InitKratosClient() in libmanager.go 
3. Added echo_factor.go::UpdateAppTokenByEmail(...)
4. Added echo_factor.go::GetUserInfoByAppToken(...). 

There are two types of tokens: authentication tokens and application tokens. Authentication tokens are managed by Kratos directly. The corresponding functions in EchoFactory should not be called anymore. App tokens are managed by echo_factor.go::GetUserInfoByAppToken(...)
